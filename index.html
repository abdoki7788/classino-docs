معرفی نرم افزار

نرم افزار مدیریت مدرسه برای آسان کردن کار مدیریت مدرسه . ساخته شده برای جشنواره خوارزمی سال ۱۴۰۱-۱۴۰۲.

کاربر های هدف

  • مدیران مدارس که برای مدیریت دانش آموزان و سازمان دادن به مدارس به یک نرم افزار نیاز دارند
  • مدیران آموزشگاه های متفرقه
  • معاونان و ناظم های مدارس

پلتفرم ها

این نرم افزار به صورت pwa در همه سیستم ها و پلتفرم ها قابل نصب و استفاده است و همچنین قابل نصب بر روی اندروید به صورت apk و در ویندوز به صورت exe است .

نحوه کارکرد کلی نرم افزار

این نرم افزار به صورت دو بخش کلاینت و سرور ساخته شده است . بخش سرور این نرم افزار با زبان برنامه نویسی پایتون و فریمورک جنگو به صورت rest api ساخته شده است و بخش کلاینت این نرم افزار با فریمورک vue.js و quasar ساخته شده و با رابط api به سرور متصل میشود.

برای مشاهده لیست اندپوینت های api به لیست اندپوینت ها مراجعه کنید و برای مشاهده لیست صفحات کلاینت به صفحه لیست صفحات مراجعه کنید .
